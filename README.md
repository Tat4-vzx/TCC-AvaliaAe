# AvaliaAê 📝

Sistema de gerenciamento de feedbacks desenvolvido em Java Swing como Trabalho de Conclusão de Curso (TCC).

## 📋 Sobre o Projeto

O **AvaliaAê** é uma aplicação desktop que permite registrar, visualizar e gerenciar feedbacks de forma simples e intuitiva, utilizando interface gráfica construída com Java Swing.

## 🚀 Funcionalidades

- Cadastro de feedbacks
- Listagem e visualização de feedbacks registrados
- Gerenciamento completo (criar, editar, excluir)
- Interface gráfica intuitiva

## 🛠️ Tecnologias Utilizadas

- **Java** — Linguagem principal
- **Java Swing** — Interface gráfica (GUI)

## ⚙️ Pré-requisitos

- Java JDK 8 ou superior instalado
- IDE recomendada: IntelliJ IDEA, Eclipse ou NetBeans

## 📦 Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/Tat4-vzx/TCC-AvaliaAe.git
```

1. Abra o projeto na sua IDE de preferência
2. Compile e execute a classe principal (`Main.java` ou equivalente)
